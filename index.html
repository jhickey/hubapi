Endpoints for Hub API

Authentication

Authentication is via your provided API key. Use the key with an auth parameter on the request. All endpoints require authentication.

/hub/feed/{user_id?}/?params=

Returns the feed. Optionally, specify a user to get just their feed. Returns JSON object. Limited to 500 items per page.

Parameters

Parameter Required Description
ch No Filter feed by a channel. Possible values are f (Facebook), t (Twitter), p (Pinterest), b (blog), e (email). Separate multiple channels by commas to combine.
ts No Only return feed items after this timestamp
sort Only if using dir Can specify time or reach to sort by these parameters. Will default to time descending if not provided.
dir No, but can only be used if sort provided Set to asc (ascending) or des (descending) to specify direction of the sort parameter.
auth Yes API key

/hub/influencer/{influencer_id}/

Updates the given influencers information as per a given JSON object. Only fileds specified in the JSON object will be altered, except the influencer_id, which is required. Returns 200 on success.

Parameter Required Description
auth Yes API key

Request Example

{
    "influencer_id": "43567542341235",
    "contact": {
        "email": "jh@branderati.com",
    },
    "geo": "Brooklyn, NY",
}

This will update the geo and email of the influencer given by the influencer_id.

Status codes

Code Description
200 Success
401 Authentication failed
422 Bad parameters or post data

/hub/tags/({influencer_id?}|{post_id?})/

Gets a list of all tags in use or a list of tags for the given post or user id.

Parameter Required Description
auth Yes API key

Response

{
  "tags": [
    "cool",
    "neato",
    "awesome"
  ]
}

An array of all tags

{
  "influencer_id": 546471341,
  "tags": [
    "cool",
    "neato"
  ]
}
Tags for a given influencer. Substitute 'influencer_id' with 'post_id' in the request URL to do the same for a given post.

Status codes

Code Description
200 Success
401 Authentication failed
422 Bad parameters

/hub/tags/({influencer_id}|{post_id})/

Adds tags given in a JSON object to the given influencer or post. Returns the item's id on success.

Parameter Required Description
auth Yes API key

Example Request

{
  "tags": [
    "cool",
    "neato"
  ]
}

Will add the tags 'cool' and 'neato' to the given influencer. Substitute 'influencer_id' with 'post_id' in the request URL to do the same for a given post.

Status codes

Code Description
200 Success
401 Authentication failed
422 Bad parameters or post data.

/hub/search/

Gets a list of influencer ids by the given search filters. See parameters below for full list of available criteria. Note that while no specific parameters are required, at least one must be present. Limited to 500 results at a time.

Parameters

Parameter Required Description
keyword No Posts containing a certain keyword.
ageLess No Age is less than given value
ageGreater No Age is greater than given value
geo With dist only. Influencers from a specific place.
dist No, but requires geo. Returns influencers within a certain number of miles from geo.
gender No Influencer's gender.
kloutGreater No Klout scores grater than this value.
friendsGreater No Facebook score friends greater than this value.
followersGreater No Twitter score followers greater than this value.
reachGreater No Total reach greater than this value.
screenerGreater No Screener score greater than this value.
pinterestGreater No Pinterest score greater than this value.
realGreater No Real Total score greater than this value.
publicGreater No Public score greater than this value.
blogGreater No BLog score greater than this value.
otherGreater No Other score greater than this value.
app No true or false, if the app is installed.
screener No true or false, if the screener sent.
pinterestGreater No Pinterest score greater than this value.
language No Specify user language.
auth Yes API key

Response

The response will be identical in format to a feed response.

Status codes

Code Description
200 Success
401 Authentication failed
422 Bad parameters.

/hub/activity/{influencer_id}

Gets an array of data from an influencers recent social media activity. Limited to 500 results per page.

Parameters

Parameter Required Description
auth Yes API key
ts No Only return results from after this timestamp.

Response

Sample response

{
    "total": "500",
    "influencer_id": "213456432",
    "next_page": "http: //branderati.com/api/hub/limit=500&offset=500&__after_id=100004539155660",
    "results": [
        {
            "type": "facebook",
            "datetime": "2013-06-19 13:00",
            "content": "This is a facebook post!",
            "interest": 3,
            "people":[
                "john.doe.51",
                "jane.smith.12",
                "joe.shmo.45"
            ]
        },
        {
            "type": "twitter",
            "datetime": "2013-6-18 02:00",
            "content": "this is a tweet!",
            "unique":{
                "hashtags": [
                    "#ahashtag",
                    "#anotherone"
                ]
            }
            }
            "interest": 2,
            "people": [
                "@johndoe",
                "@janedoe"
            ]
        },
    ...
    ]
}

Response breakdown

Key Description
total total number of posts returned
influencer_id ID of the specific influencer
next_page Call to next page of results. Will be false if there are no more pages left.
results An array of objects representing the influencer's posts.
results.type The social media network of the post.
results.datetime Time of the posting in YYYY-MM-DD HH:MM
results.content The text content, if any, of the post
results.unique An array of any specific data unique to the post type, such as hash tags for a Twitter post.
results.interest This represents the number of people who responded to the post in the way relevant to the social network in question. For example, for Facebook, this represents 'likes', for Twitter it is retweets.
results.people The social media handles of the people who showed interest in the post.

Status codes

Code Description
200 Success
401 Authentication failed
422 Bad parameters
